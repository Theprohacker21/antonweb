## How to Use

1. Open the website at `http://localhost:3000`
2. GeForce NOW will load automatically
3. Sign in with your NVIDIA account
4. Browse and play your games!

## Requirements

- Active internet connection
- NVIDIA account
- Supported device (PC, Mac, mobile, tablet)
- GeForce NOW subscription (Free or paid)

## Files

- `index.html` - Main page with GeForce NOW iframe
- `style.css` - Styling for full-screen display
- `script.js` - Minimal JavaScript for iframe handling
- `server.js` - Express backend server
- `package.json` - Dependencies

## Installation

```bash
npm install
npm start
```

Then visit `http://localhost:3000` in your browser.

## Notes

- GeForce NOW must be publicly accessible for the iframe to load
- Some network restrictions may prevent iframe loading - try accessing geforceNow.com directly if needed
- The service requires an active NVIDIA GeForce NOW account
